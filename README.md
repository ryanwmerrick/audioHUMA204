# Classical Audio Quiz

A simple static web app that plays a random segment from one of your audio files and asks the user to identify the correct title.

## How to use

1. Open `index.html` in a browser.
2. Choose the clip length.
3. Click **Start Quiz**.
4. Click **Play Clip** to hear the random segment.
5. Choose the matching title.
6. Click **Next Clip** to continue.

## Project files

- `index.html` — quiz interface
- `style.css` — visual styling
- `script.js` — quiz logic and random clip playback
- `audio/` — your audio files

## Notes

- The page uses the browser audio element and random start positions.
- For best results, open the page from a local web server or use a browser that permits local audio playback from file URLs.
